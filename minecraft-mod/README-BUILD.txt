========================================================
   HOW TO BUILD AXMED555 VISUALS MOD ON YOUR PC
========================================================

WHAT YOU NEED:

1. INSTALL JAVA 8
   Link: https://adoptium.net/temurin/releases/?version=8
   - Download .msi file for Windows
   - Install (Next -> Next -> Finish)
   - RESTART YOUR COMPUTER!

2. EXTRACT THIS ZIP FILE
   - Right-click the ZIP
   - Select "Extract All"
   - Open the "minecraft-mod" folder

3. RUN THE BUILD SCRIPT
   - Open the "minecraft-mod" folder
   - Double-click "BUILD-MOD.bat"
   - Wait 10-15 minutes (first time only!)
   - Do NOT close the window!

4. GET YOUR MOD
   - After "BUILD COMPLETE" your mod is ready!
   - Location: build\libs\axmed555-visuals-1.0.0.jar

5. INSTALL IN MINECRAFT
   - Install Forge 1.16.5 (version 36.2.39+)
   - Copy JAR to .minecraft\mods\
   - Launch the game!

========================================================

TROUBLESHOOTING:

Q: Window closes immediately?
A: You need to RESTART your computer after installing Java!

Q: "gradlew.bat not found"?
A: You are running the file from the wrong folder!
   Open the "minecraft-mod" folder first!

Q: Takes too long?
A: First build ALWAYS takes 10-15 minutes!
   Gradle downloads many files from the internet.
   Next builds will take only 30 seconds!
